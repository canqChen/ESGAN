# ESGAN-tensorflow
A simple tensorflow implementation of ESGAN that can transfer image emotion
## Requirement
* Tensorflow 1.9
* python 3.6
* scipy
## Usage
### Train
* bash run_train.sh
### Test
* bash run_test.sh
